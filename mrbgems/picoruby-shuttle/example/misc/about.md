# About This Blog

Welcome to my PicoRuby blog! This site is generated using **PicoRuby Shuttle**, a static site generator powered by PicoRuby WASM.

## What is PicoRuby?

PicoRuby is a lightweight Ruby implementation designed for embedded systems and WebAssembly. It brings the elegance of Ruby to resource-constrained environments.

## About Shuttle

Shuttle is a static site generator that runs entirely in the browser using PicoRuby WASM. It features:

- **Query string-based routing** for SEO-friendly URLs
- **Markdown support** with front matter
- **GitHub Pages compatibility**
- **Customizable design** with Tailwind CSS
