BLE UART Demo

IO-compatible UART interface over BLE.
Uses JS::BLE::UART with configurable UUIDs.

LightBlue (iOS/Android) Peripheral Setup
  1. Open LightBlue → tap Virtual Devices tab
  2. Tap + to create a new Virtual Peripheral
  3. Add a Service with UUID 0xFFE0
  4. Add a Characteristic under that service with UUID 0xFFE1
  5. Enable these properties on the characteristic:
    • Read — allows Central to read values
    • Write Without Response — required for write() / puts() from browser
    • Notify — allows Central to receive data via read() / gets()
  6. Start advertising, then click Connect above
  7. To send data from LightBlue to browser: tap the characteristic → write a value → it will appear in the Log via Notify
